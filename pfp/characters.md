# Characters — Canon Base Archetypes

The base character is determined by the interview. The first question gates the available pool by gender. All characters are from VVC canon.

Base images live at `Meme Factory UI/suite/public/assets/chars/`. Paths in the table are relative to that folder.

---

## Question 1: Are you male, female, or rather not say?

This single answer determines which character pool the rest of the interview draws from.

## Asset Notes

**Canonical base image folder:** `Meme Factory UI/suite/public/assets/chars/`

All 27 roster characters have a PNG there. Non-obvious filename mappings:
- `chad.png` = Chad (source: `BSC (SOL)/images/chad.png`)
- `basdchad.png` = unassigned
- `gigachad.webp` = Gigachad (source: `BSC (SOL)/images/Gigachad2.webp`)
- `boomer.png` = Dad

Also in the folder but not in this roster: `lshad.png`, `wraith.png` — available for future expansion.

## Design Notes

- **Deity tier as "rather not say" incentive:** Assigning the highest rarity to the non-binary pool rewards honest answers and makes "rather not say" a desirable outcome rather than a fallback.
- **Character recognition rule:** The base archetype must be immediately legible. A Chad with a top hat and tattoos is still unmistakably a Chad. Attributes cannot override the core visual identity.
